Exercise 2.2
Create a particle system – initial setup
  • The initial code contains a vertex buffer containing only the (x,y) position for each particle in normalized device coordinates. The shader also contains a shader attribute for the coordinates (vec2).
  • Add velocity (float x,y) and timeOfBirth (float) to the vertex buffer. This means that each vertex should have a position (x,y), a velocity (x,y) and timeOfBirth (float).
  • Modify the emitParticle such that adds velocity and timeOfBirth to the vertexBuffer.
  • Add velocity and timeOfBirth as vertex attributes to the shader (leave them unused for now).
  • Verify that the code still runs without errors and shows the initial scene.
Exercise 2.3
Create a particle system – moving particles
  • Create a float uniform in the vertex shader called currentTime in the shader and set the value in the render loop.
In the vertex shader:
  • If the timeOfBirth == 0 set the particle position outside the view frustum. This will hide the uninitialized particles (the white dot in the middle of the screen).
  • Else if timeOfBirth > 0
    • Compute the age of the current particle;
    • Compute the current position of the particle by adding the age*velocity;
  • The maximum age for the particles should be 5 seconds. If a particle is older than maximum age, then move it outside the view frustum.
  • You should now see particles move around when drawing.
Exercise 2.4
Create a particle system – Adding alpha blending
  • Enable alpha blending in OpenGL by adding to following before the render loop:
    glEnable(GL_BLEND);
    glBlendFunc(GL_SRC_ALPHA, GL_DST_ALPHA);
  • This means that the particles are now blended together (based on the alpha in the color).
  • Set the alpha value 0.2.

Exercise 2.6
Create a particle system – Improving the particles appearance part 2 A commonly used function in GLSL is mix(x,y,a), which interpolates linear between x and y based on the value a (returns x * (1-a) + y * a).
  • Let particles size grow from 0.1 at the time of birth to 20.0 at the time of death (maximum age).
  • Change maximum age to 10.0.
  • Send the age to the fragment shader using the out and in variable qualifiers.
  • The particles slowly become transparent as they get older (fully transparent at the time of death).
  • Modify the color of the particle based on the age such that it smoothly interpolates between the colors:
    • (1.0,1.0,0.05) (Time of birth);
    • (1.0,0.5,0.01) (At the midpoint of their life);
    • (0.0,0.0,0.00) (Time of death).
